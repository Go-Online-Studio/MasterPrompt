# Master Prompt Project

This project contains two portable HTML prompt pages:

- `index.html` - use at the start of a website build.
- `final-analyser.html` - use near launch to audit and fix SEO, GEO, speed, accessibility, and security issues.

Open either file directly in a browser. Each page has copy and download buttons for the full prompt text.

## Main Changes From The Original Prompts

- Split creation work from final audit work.
- Added a mandatory clarification gate before coding.
- Removed hard-coded Bootstrap/GSAP assumptions from the starter prompt.
- Moved deployment, redirects, vulnerability checks, dependency audits, and final production cleanup to the analyser prompt.
- Added GEO and AI-agent visibility guidance while avoiding false promises about guaranteed AI search inclusion.
